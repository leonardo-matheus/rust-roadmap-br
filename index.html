01

Introdução à Programação e ao Rust

O que é programação?

Programar é como ensinar um computador a executar tarefas, assim como ensinar alguém a seguir uma receita de bolo. Você fornece instruções detalhadas e o computador as executa exatamente como foram dadas.

Por que aprender programação?

A programação desenvolve o raciocínio lógico, abre portas para o mercado de trabalho e permite criar soluções para problemas do dia a dia.

O que é Rust?

Rust é uma linguagem de programação moderna, criada para ser segura, rápida e eficiente. Imagine um carro esportivo com todos os itens de segurança: desempenho e proteção juntos.

Diferenças entre Rust e outras linguagens

Rust combina a velocidade do C/C++ com a segurança de linguagens modernas como Python.

Primeiros Passos

Instalando o Rust

Windows

Baixe o instalador em rustup.rs

Linux/Mac

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Seu primeiro programa

fn main() {
    println!("Olá, mundo!");
}

Esse comando imprime uma mensagem na tela. Pense nele como acender a luz de um quarto para mostrar que tudo está funcionando.

Ownership & Borrowing

Imagine que cada valor é uma bola. Só pode haver um dono por vez. Quando você passa a bola, perde o direito de usá-la.

Borrowing
